About Neural Studio

 Neural Studio is an educational software that will be able to show the structure of common neural networks using educational datasets like Iris and MNIST. Neural Studio will be able to show the effect of adding neurons and hidden layers to sequential, convolutional, and recurrent neural networks. Neural Studio will also give the user the ability to see the loss graph, which will show if the model is under or over trainined and can show how this effects the accuracy of the model.
