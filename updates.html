Project Updates

Prototype and Future Development

  This week we completed work on our prototype to demo the software. The prototype includes the basic startup menu for the software and the current UI for the Sequential Neural Network visualization. While there is still work to be done on this module the prototype highlights features like updating neuron weights and the variable construction of the network from user input.
  Future development based on this prototype is centered on some key feedback from our sponsor. We are looking to add some live update features for training loss to provide information to discuss with students while training is still executing. We are also looking to improve the data sent to the UI by including additional graphs and output data from training.

Design Document 2 and Preparing for Prototype Demo!

 This week our team finsihed our second design document. As opposed to the first design document, this document went into further detail of how Neural Studio works using data flow diagrams and psuedocode.
 Along side the second design document we have been preparing for our first prototype demonstration. The coming week will involve making sure that the team is on the same page in terms of the main points of the project we want to highlight in our demonstration.
  On the development side of the project, we are continuing to finish up the convolutional neural network and will soon start the final aspect of Neural Studio. The recurrent neural network!!

First Sponsor Demo & Feedback!

 With the first demo finished and able to be tested by our sponsor on Friday 3/15 we were able to get some much needed feedback on the further direction of our software! Ontop of the sponsor demo, we are currently working on a high-level design document that will be used to explain the functionality of our software using the most general language possible to allow for interpretation and recreation. The next important date to look forward to will be a full prototype to show to our Professor just before we put the finishing touches on our functionality!

03/11/2024

Development of Convolutional Neural Network

 The team has begun creating a design document for the project and will be preparing for the first demo to our project advisor shortly after its completion.
  Development of the GUI for the IRIS dataset is finishing up. The neural network has been connected to the front-end GUI and is able to accurately show training weights and outputs. There are still several features that we would like to incorporate as the project developes, but we have the GUI working with its key functionality. Our current focus is now on incorporating a convolutional neural network for the MNIST dataset. Backend development of the network itself is largely finished but visualization is only in the beginning phases.

02/09/2024

First steps in creating Neural Studio

 As of today, we have completed the draft of the Software Requirements Specification document, created our GitHub page, and began initial planning of the project. We have talked to Dr. Butler (a data science professor) who has some experience in building neural networks to gain a better understanding of the end goal. We have also began making some initial progress in building out the sequential neural network for the Iris dataset. Overall, these last couple weeks have been crucial to the beginning of the project and setting a clear goal that we will aim to achieve by the end of the deadline. Our current next steps are getting our Kanban backlog up to date and continue making progress on the code and structure of the program itself.
