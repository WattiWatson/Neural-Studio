Project Updates

Development of Convolutional Neural Network

 The team has begun creating a design document for the project and will be preparing for the first demo to our project advisor shortly after its completion.   Development of the GUI for the IRIS dataset is finishing up. The neural network has been connected to the front-end GUI and is able to accurately show training weights and outputs. There are still several features that we would like to incorporate as the project developes, but we have the GUI working with its key functionality. Our current focus is now on incorporating a convolutional neural network for the MNIST dataset. Backend development of the network itself is largely finished but visualization is only in the beginning phases.

02/18/2024

Planning Finished! Development of Dataset 1 is nearing completion.

 This week we were able to finish our schedule and continue working through the development on the project. Also, exciting news, we released our first press release poster! We have prepared a presentation for the official announcement of Neural Studio. This will be presented around 3 p.m ET.
 Within the development aspect of the project we have taken major steps towards the completion of the Iris Dataset. The backend Python code has been completed and the visualization side is nearing completion. Soon we will be able to start on the MNIST dataset! The beginning of the development of our home page has also been started. This will allow us for users to navigate between samples. With a lot of work still to be done there is no time to waste. More information will be given next week in the next update!

02/18/2024

Planning Continues: Development Begun!

 After finishing our final draft of our Software Requirements Doc, meetings this week were mostly on topics of project distribution and getting everyone set up to work on code development. We have begun work on our Kanban Schedule through Github and have begun the draft of our first Press Release Poster (coming soon).
 On the development side, the Iris dataset is done being implemented into our program, and we are in the beginning steps of creating the framework of our visualization(2D) GUI. The visualization is looking promising, and we are currently working on the challenge of combining the visualization and the first dataset! Progress on the second dataset, MNist, will be started shortly, though its already looking to be more challenging than the first.

02/09/2024

First steps in creating Neural Studio

 As of today, we have completed the draft of the Software Requirements Specification document, created our GitHub page, and began initial planning of the project. We have talked to Dr. Butler (a data science professor) who has some experience in building neural networks to gain a better understanding of the end goal. We have also began making some initial progress in building out the sequential neural network for the Iris dataset. Overall, these last couple weeks have been crucial to the beginning of the project and setting a clear goal that we will aim to achieve by the end of the deadline. Our current next steps are getting our Kanban backlog up to date and continue making progress on the code and structure of the program itself.
